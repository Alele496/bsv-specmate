# ⚠️ Bluespec Classic (.bs) — 旧语法，仅供参考

此目录包含 BSC 测试套件中的 882 个 `.bs` 文件（Bluespec Classic 语法）。

## 为什么在这里

- 部分实现模式仅在 `.bs` 中有示例，`.bsv` 中可能找不到
- 可用于参考算法思路和模块结构

## 使用原则

| ✅ 可以参考 | ❌ 不要直接使用 |
|------------|---------------|
| 模块划分思路 | 语法（如 `~` 对 Bool） |
| 状态机设计 | 已废弃的写法 |
| 调度逻辑 | 缺少 provisos 的旧写法 |

## 新旧语法差异（常见陷阱）

| 旧语法 (.bs) | 新语法 (.bsv) |
|-------------|--------------|
| ~~`x = ~x` (Bool)~~ | `x = !x` |
| ~~旧版标准库 API~~ | 当前 bsc 2025.07 标准库 |
| 可能缺少 `provisos` | 必须声明类型约束 |

> **Agent 注意**：如果 AI 训练数据较旧，可能会倾向使用 .bs 语法或旧版 API。遇到语法错误时优先到 `examples/bsv/` 找正确写法。
