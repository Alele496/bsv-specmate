定位

specmate 是什么——以及它不是什么

噪声诅咒：BSV 硬件设计中，许多静态检查工具的误报率居高不下。每一次误报都在消耗开发者的信任——就像"狼来了"。specmate 选择了完全不同的路线：用轻量正则 + tree-sitter 快速筛查可疑模式，但永远把最终裁决权交给 BSC 编译器。不做 BSC 已经在做的事，只做 BSC 不做的。
