# T0061 — Bool 不能使用位操作符 (×1)

**bsc 输出**：
```
Error: "...", line 88, column 25: (T0061)
  Bitwise operators are not defined for Boolean values. Consider using the
  Boolean operators && and || instead.
```

**原因**：`Bool` 是逻辑类型（非 `Bit#(n)`），不能使用位操作符。

**解决**：用逻辑操作符替换。

```bsv
// 错误
Reg#(Bool) wave <- mkReg(False);
wave <= ~wave;     // Bool 不能用 ~

// 正确
wave <= !wave;
```

| 类型 | 操作符 |
|------|--------|
| `Bool` | `!` `&&` `\|\|` `==` `!=` |
| `Bit#(n)` | `~` `&` `\|` `^` `==` `!=` |

> **规则**: Bool 用逻辑操作符，Bit#(n) 用位操作符。两者不混淆。
