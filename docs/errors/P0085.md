# P0085 — 属性语法错误 (x2)

## 子情况 1：synthesize 拼写错误

**bsc 输出**：
```
Error: "Top.bsv", line 8, column 5: (P0085)
  Unknown attribute "synthesized". Did you mean "synthesize"?
```

**原因**：`(* synthesize *)` 写成了 `(* synthesized *)`——这是 BSC 属性系统中最常见的拼写错误。`synthesized` 不是合法的 BSV 属性，编译器不认识。

**解决**：改为 `synthesize`。

```bsv
// 错误
(* synthesized *)               // ❌ 拼写错误
module mkTop(TopIfc);
    ...
endmodule

// 正确
(* synthesize *)                // ✅
module mkTop(TopIfc);
    ...
endmodule
```

## 子情况 2：同一 `(* *)` 块中属性重复

**bsc 输出**：
```
Error: "Top.bsv", line 10, column 5: (P0085)
  Attribute "descending_urgency" is repeated in the same attribute list.
```

**原因**：在同一个 `(* *)` 括号内相同属性名出现了多次。

**解决**：删除重复的属性声明。

```bsv
// 错误
(* descending_urgency = "rl_a, rl_b", descending_urgency = "rl_c, rl_d" *)
//                    ~~~~~~~~~~~~~~~~   ~~~~~~~~~~~~~~~~

// 正确
(* descending_urgency = "rl_a, rl_b, rl_c, rl_d" *)
```

## 规则

> BSV 属性名不能拼错（尤其 `synthesize` 不写作 `synthesized`）。同一 `(* *)` 块中每个属性名只能出现一次。多行属性写在同一个 `(* *)` 或用多个 `(* *)` 块分隔。
