# T0030 — 多态模块无法 synthesize (×1)

**bsc 输出**：
```
Error: "StreamBuf.bsv", line 12, column 8: (T0030)
  Cannot synthesize `mkStreamBuf': Its interface is polymorphic.
```

**原因**：BSV 的 `(* synthesize *)` 只能应用于具体类型的模块。带类型参数（`#(type t, numeric type depth)`）的多态模块无法直接 synthesize——bsc 需要知道所有类型的具体值才能生成 Verilog。

**解决**：添加具体类型的 non-polymorphic wrapper。

```bsv
// 错误：多态模块带 synthesize
(* synthesize *)
module mkStreamBuf(StreamBuf#(t, depth));   // t 和 depth 是类型参数
    ...
endmodule

// 正确：添加具体类型 wrapper
(* synthesize *)
module mkStreamBuf8(StreamBuf#(Bit#(8), 8));  // 具体类型
    let m <- mkStreamBuf;
    return m;
endmodule

// 在 Top 中使用 wrapper
StreamBuf#(Bit#(8), 8) f <- mkStreamBuf8;
```

> **规则**: 多态模块不能带 `(* synthesize *)`。需要 synthesize 时创建具体类型的 wrapper 模块。
