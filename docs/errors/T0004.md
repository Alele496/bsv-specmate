# T0004 — Unbound variable (×1)

> 适用版本: BSC 2025.07 | 自动生成: 2026-07-15 | 来源: 1 captures

**bsc 输出**：
```
Error: "WbArbiter.bsv", line 54, column 48: (T0004)
  Unbound variable `vec'
```

**原因**：

`vec()` 不是 BSC 标准库 `Vector::*` 的导出函数。虽然部分 Bluespec 教程和旧版文档中提到了 `vec()` 作为 Vector 构造语法，但在 BSC 2025.07 中，标准库不导出此函数。

**解决**：

用 `genWith` 替代：

```bsv
// 错误
Vector#(4, Bit#(2)) pri <- mkReg(vec(0, 1, 2, 3));

// 正确
Vector#(4, Bit#(2)) pri <- mkReg(genWith(fromInteger));

// 或显式写出索引
Vector#(4, Bit#(2)) pri <- mkReg(genWith(
    function Integer i; return fromInteger(i); endfunction
));
```

**相关**：`req_vec()` 函数中构造 `Vector#(4, Bit#(1))` 同理需用 `genWith`：

```bsv
// 错误
return vec(m0_req, m1_req, m2_req, m3_req);

// 正确
function Bit#(1) req_at(Integer i);
    return case (i)
        0: m0_req; 1: m1_req;
        2: m2_req; 3: m3_req;
    endcase;
endfunction
return genWith(req_at);
```

> **规则**: BSV 中构造 Vector 用 `genWith` 而非 `vec`。`genWith` 在 `Vector::*` 中始终可用。
