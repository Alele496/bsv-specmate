# G0053 — 子模块初始化用了动态表达式

## 现象

```
Error: "Top.bsv", line 58, column 19: (G0053)
  Parameter `init' of submodule `half_cyc' is instantiated with a dynamic
  expression. Instantiation parameters must have static values.
```

## 原因

`mkReg(init_val)` 的 `init_val` 必须是编译期静态常量（字面量如 0、1，或 `?`）。模块参数如 `Bit#(1) cpol` 在 BSV 中不被视为编译期静态常量——因为相同的模块可被不同的参数值实例化。

## 解决方案

改用 `mkRegU`（未初始化寄存器），然后在 rule 或 method 中显式赋值。

```bsv
// 错误 ❌
Reg#(Bit#(1)) sck_out <- mkReg(cpol);

// 正确 ✅
Reg#(Bit#(1)) sck_out <- mkRegU;
// 在 start method 或 rule 中：sck_out <= cpol;
```

## 规则

> 寄存器初始化只能用字面量（0, 1, ?, maxBound）或编译期常量。模块参数不在此列 → 用 mkRegU + 后续显式赋值。
