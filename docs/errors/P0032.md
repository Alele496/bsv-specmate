# P0032 — Methods must be at end of block (×2)

> 适用版本: BSC 2025.07 | 自动生成: 2026-07-15 | 来源: 2 captures

**bsc 输出**：
```
Error: "...", line 77, column 5: (P0032)
  Methods and subinterfaces must be at end of block
```

**原因**：

BSV 模块内 rule 必须全部写在 method 之前。只要有 method 出现在 rule 之前就报错。

**解决**：

将所有 method（含 Action 和 value method）统一移到所有 rule 之后。

```bsv
// 错误
module mkFoo(Foo);
    method Action bar(...); ... endmethod   // ← method 在前
    rule rl_a; ... endrule                  // ← 触发 P0032
endmodule

// 正确
module mkFoo(Foo);
    Reg#(Bit#(8)) x <- mkReg(0);
    rule rl_a; ... endrule
    rule rl_b; ... endrule

    method Action bar(...); ... endmethod
    method baz = ...;
endmodule
```

> **规则**: 每个模块内，rule 全部在前，method 全部在后。中间不能交替。
