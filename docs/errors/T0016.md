# T0016 — Struct 字段不存在 (x2)

> 适用版本: BSC 2025.07 | 自动生成: 2026-07-15 | 来源: 2 captures

## 子情况 1：struct 字面量使用了不存在的字段名

**bsc 输出**：
```
Error: "Uart.bsv", line 90, column 18: (T0016)
  The field "baudrate" does not belong to the type "UartConfig".
  Valid fields: baud_div, data_bits, parity_en, stop_bits
```

**原因**：

struct 字面量中引用的字段名不在该 struct 的定义中。BSV 类型检查会验证所有字段名。

**解决**：

检查 struct 定义中的字段名，使用正确的名称。

```bsv
// 结构体定义
typedef struct {
    Bit#(16)  baud_div;
    Bit#(2)   data_bits;
    Bool      parity_en;
    Bit#(1)   stop_bits;
} UartConfig;

// 错误 — 字段名 "baudrate" 不存在
UartConfig cfg = UartConfig {
    baudrate: 16'd1152,     // ❌ 应该是 baud_div
    data_bits: 2'd3,
    parity_en: False,
    stop_bits: 1'd1
};

// 正确
UartConfig cfg = UartConfig {
    baud_div: 16'd1152,
    data_bits: 2'd3,
    parity_en: False,
    stop_bits: 1'd1
};
```

## 子情况 2：用错了 struct 类型

**原因**：

字段名存在但属于另一个 struct，当前 struct 字面量用了错误的类型名。

**解决**：

确认 struct 字面量的类型名与目标类型一致。

```bsv
// 错误 — SendConfig 和 RecvConfig 是不同的 struct
SendConfig cfg = RecvConfig {     // ❌ 类型名写错
    timeout: 16'd100,
    retry: 2'd3
};

// 正确
SendConfig cfg = SendConfig {
    timeout: 16'd100,
    retry: 2'd3
};
```

## 规则

> struct 字面量的类型名和字段名必须与 struct 定义完全一致。BSV 中 struct 字段名区分大小写。
