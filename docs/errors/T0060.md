# T0060 — 寄存器位宽不匹配 (×1)

**bsc 输出**：
```
Error: "...", line 87, column 63: (T0060)
  Expression could not be compiled because of a bit-size mismatch.
  The type: Bit#(1)
  has a bit size of 1 but is either being used as or created by a bit vector
  of size 2.
```

**原因**：拼接表达式 `{...}` 总位宽与目标寄存器不匹配。BSV **可能不直接报 "16 vs 15"，而是报子元素（如 2 位→1 位）的冲突。**

**解决**：修改目标寄存器位宽为正确值。

```bsv
// 错误：shift_reg 是 15 位，但拼接结果是 16 位
Reg#(Bit#(15)) shift_reg <- mkReg(0);
// {cmd_ch, 2'b10, cmd_val, 2'b00} → 16 位

// 正确
Reg#(Bit#(16)) shift_reg <- mkReg(0);
```

> **规则**: T0060 的报错行可能指向拼接子元素而非目标寄存器。优先检查赋值目标的位宽。

> 位宽扩大时，相关寄存器同步扩宽 → 见 T0051。
