# T0043 — 模块包含无法综合的类型参数 (×1)

> 适用版本: BSC 2025.07 | 自动生成: 2026-07-15 | 来源: 1 captures

**bsc 输出**：
```
Error: "Top.bsv", line 12, column 8: (T0043)
  The type of the synthesized module's interface is polymorphic.
  Type variable(s) `_1' cannot be determined.
```

**原因**：

BSV 的 `(* synthesize *)` 模块的所有参数必须是 `Bits` 类的具体类型（如 `Bit#(n)`、`UInt#(n)`），不能有 `Integer` 参数、`numeric type` 自由变量或 `provisos`。带 `provisos` 的多态接口不能被 synthesize——bsc 需要知道所有类型的具体值才能生成 Verilog。

常见触发形式：
- 模块参数用 `Integer` 而非 `Bit#(n)`：`module mkUartTx#(Integer baud_div)`
- 接口含 `numeric type`：`interface SpiMaster#(numeric type data_width)`
- 模块带 `provisos` 约束

**解决**：

将 Integer 参数改为 `Bit#(n)` 或 `UInt#(n)` 的具体类型，去掉多态接口。

```bsv
// 错误 ❌ — Integer 参数不可综合
(* synthesize *)
module mkUartTx#(Integer baud_div)(UartTx);
    ...
endmodule

// 正确 ✅ — 用 Bit#(16) 具体类型
(* synthesize *)
module mkUartTx#(Bit#(16) baud_div)(UartTx);
    ...
endmodule

// 错误 ❌ — 多态接口不可综合
interface SpiMaster#(numeric type data_width);
    method Action send(Bit#(data_width) data);
endinterface

// 正确 ✅ — 去掉 numeric type，用固定位宽
interface SpiMaster;
    method Action send(Bit#(8) data);
endinterface
```

> **规则**: `(* synthesize *)` 模块的所有参数和接口必须是 `Bits` 类的具体类型。模块参数用 `Bit#(n)`/`UInt#(n)` 而非 `Integer`，接口不用 `numeric type` 或 `provisos`。
