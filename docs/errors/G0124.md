# G0124 — BVI Method 名与 Verilog Port 名不匹配

## 现象

```
Error: "Top.bsv", line 55, column 8: (G0124)
  BVI method name "spi_clk" does not match any Verilog port in the imported module.
  Expected port names: CLK, MOSI, MISO, CS_N
```

## 原因

在 `import BVI` 块中，`method` 声明后面的**引号内名称**必须与 Verilog 模块的端口名**完全一致**（包括大小写和下划线）。BSV 编译器在生成 Verilog 时会用这个名称连接端口——名称对不上就没法连。

常见触发：
- Verilog 端口名全大写（`CLK`），BVI 中写成小写（`clk`）
- 端口名含下划线位置记错（`data_in` vs `data_in_`）
- 从其他模块的 BVI 声明复制过来忘记改 method 端口名

## 解决方案

**检查 Verilog 源文件中的 module 端口列表，确保 BVI 中的每个 method 引号内名称完全对应。**

```bsv
// 假设 Verilog 模块端口为：
// module spi_slave (input SCK, input MOSI, output MISO, input CS_N);

// 错误 — 大小写不匹配
import "BVI" spi_slave =
module mkSpiSlave#(...)(SpiSlaveIfc);
    method sck sck() enable((*inhigh*) EN_sck);           // ❌ "sck" ≠ "SCK"
    method mosi mosi() enable((*inhigh*) EN_mosi);        // ❌ "mosi" ≠ "MOSI"
    method miso miso() enable((*inhigh*) EN_miso);        // ❌ "miso" ≠ "MISO"
    method cs_n cs_n() enable((*inhigh*) EN_cs_n);        // ❌ "cs_n" ≠ "CS_N"
endmodule

// 正确 — 大小写完全匹配 Verilog 端口名
import "BVI" spi_slave =
module mkSpiSlave#(...)(SpiSlaveIfc);
    method sck SCK() enable((*inhigh*) EN_sck);           // ✅
    method mosi MOSI() enable((*inhigh*) EN_mosi);        // ✅
    method miso MISO() enable((*inhigh*) EN_miso);        // ✅
    method cs_n CS_N() enable((*inhigh*) EN_cs_n);        // ✅
endmodule
```

## 规则

> BVI `method` 声明中的端口名（引号内名称）必须与 Verilog 源文件的 module 端口 **100% 逐字符匹配**。大小写、下划线全部精确对齐。
