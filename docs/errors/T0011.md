# T0011 — 方法名与寄存器名冲突 (×1)

> 适用版本: BSC 2025.07 | 自动生成: 2026-07-15 | 来源: 1 captures

**bsc 输出**：
```
Error: "Uart.bsv", line 121, column 5: (T0011)
  Unresolved overloading — method name collides with register name.
```

**原因**：

模块内方法名和寄存器名相同时，BSV 编译器无法消歧义。
当 `method Bool flag = flag;` 中的 RHS `flag` 既可以指向寄存器（`_read`）也可以指向方法自身（递归），编译器报 T0011。

**解决**：

寄存器改名，方法名保留（对外接口不变）。

```bsv
// 错误
Reg#(Bool) rx_overflow <- mkReg(False);      // 寄存器
method Bool rx_overflow = rx_overflow;        // 方法名冲突

// 正确
Reg#(Bool) over_flag <- mkReg(False);         // 寄存器改名
method Bool rx_overflow = over_flag;           // 方法不变
```

> **规则**: 寄存器名不要和方法名重名。寄存器加后缀 `_reg` 或 `_r`、方法用对外接口名。
