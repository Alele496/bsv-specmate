# P0030 — Value method 语法错误 (×1)

> 适用版本: BSC 2025.07 | 自动生成: 2026-07-15 | 来源: 1 captures

**bsc 输出**：
```
Error: "Uart.bsv", line 202, column 23: (P0030)
  Method tx is a value method but uses action syntax.
```

**原因**：

BSV 中 `method Type name; ... endmethod` 是一个 Action 上下文。在 value method（非 Action 方法）中使用 `if-return` 会生成 `ActionValue` 而非纯值，编译器报 P0030。

**解决**：

value method 必须用 `=` 短语法，多条件用 `? :` 三元链或提取 `function`。

```bsv
// 错误
method Bit#(1) tx;
    if (!tx_busy) return 1'd1;              // if-return = Action context
    if (tx_state == 4'd1) return 1'd0;
    return 1'd1;
endmethod

// 正确：三元链
method Bit#(1) tx = (!tx_busy)                     ? 1'd1 :
                     (tx_state == 4'd1)             ? 1'd0 :
                     (tx_state >= 4'd2 && ...)      ? tx_shift[0] :
                     1'd1;
```

**或提取为 function**：

```bsv
function Bit#(1) tx_val();
    if (!tx_busy) return 1'd1;
    if (tx_state == 4'd1) return 1'd0;
    return 1'd1;
endfunction
method Bit#(1) tx = tx_val;
```

> **规则**: BSV 中 `method Type name = expr;` 是 value method（纯组合逻辑）。
> `method Action name(...); ... endmethod` 是 Action method（可改状态）。
> value method 不可用 `if-return-endmethod` 语法。
